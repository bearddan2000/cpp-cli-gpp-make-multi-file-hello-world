# cpp-cli-gpp-make-multi-file-hello-world

## Description
This is a POC project to demonstrate a
make as a build tool.

## Tech stack
- g++
- make
- c++

## Docker stack
- alpine:edge

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
